# setup

## requirements

|        |        |
| ------ | ------ |
| `node` | `>=20` |
| `yarn` | `1.x`  |

## install

1. clone repository

```sh
git clone <repository-name>
```

2. install `node` dependencies

```sh
yarn
```

---

# commands

### development

```sh
yarn start
```

### build

```sh
yarn build
```

### nix-shell environment

```sh
nix-shell env.nix
```
